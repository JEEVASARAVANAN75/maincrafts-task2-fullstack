# 🌐 Multi-Page Website | Full Stack Internship Task 2

This project is developed as part of the **Full Stack Web Development Internship – Task 2** at Maincrafts Technology.

## 📌 Project Overview
The objective of this task is to build a **multi-page website** with proper navigation and form validation. The website includes Home, About, and Contact pages, simulating a real-world business website.

## ✨ Features
- 🔹 Multi-page navigation (Home, About, Contact)
- 🔹 Responsive design (Mobile & Tablet)
- 🔹 Contact form with JavaScript validation
- 🔹 Clean and modern UI
- 🔹 FontAwesome icons integration

## 🛠 Technologies Used
- HTML5
- CSS3
- JavaScript
- FontAwesome (CDN)

## 📁 Project Structure

project/
│── index.html
│── about.html
│── contact.html
│── style.css
│── script.js

## 🚀 How to Run
1. Download or clone the repository
2. Open `index.html` in your browser
